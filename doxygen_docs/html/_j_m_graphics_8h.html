Detailed Description

An OpenGL based graphics object that can draw various shapes, meshes, text and images.

Each instance of this object opens one window. Each frame must start with a beginDraw() function and end with an endDraw() function.

Example of a minimal draw loop:

JMGraphics* gr = new JMGraphics(1920, 1080, "My window"); while(gr->windowOpen()) { gr->beginDraw(); //draw stuff here gr->endDraw(); } delete(gr);

Aditional glfwWindowHints may be passed before initialization if reqired.
